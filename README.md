# Essence

What if there were a game like Zoo Tycoon, but you could _attack other zoos_?
Essence combines creature taming, combat, and terraforming, bringing together the best core features of games like Pokemon, Zoo Tycoon, and Terraria.
Creatures and land forms replace the collecting of items that is core to other RPGs—tame creatures with new abilities to advance the game.

View screenshots on my [gamedev portfolio page](https://uncannyforest.com/projects) or the [essence game](https://blog.uncannyforest.com/tagged/Essence%20game) tag on my blog.

## Implementation

3D and 2D assets are integrated seamlessly with billboarding.  Gameplay is on a 2D isometric grid.

NPC AI uses a hand-coded state management system managing update messages, similar to Redux.

Terrain procedurally generated using cellular automata.

Built with Unity 2020.3.14f1.
